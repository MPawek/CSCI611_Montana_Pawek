# Assignment 3 - YOLO: How to use this code

## Install Dependencies
pip install ultralytics opencv-python

## Prepare Data
Make sure data is converted to YOLO format with convertToYolo.py. You'll need to modify the paths in it to link to your own raw image input, annotated list input, and output directory. This file was generated by ChatGPT to save time.
python convertToYolo.py

## Train Model
Make sure paths in trainModel.py link to the prepared data.
python trainModel.py

## Run Baseline YOLO
python evaluate_baseline.py

## Run Trained Model
Make sure path to best.pt is correct in the script.
python evaluate_trained.py
