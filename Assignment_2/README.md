# CNN Image Classification -- Assignment 2
This project implements a Convolutional Neural Network in PyTorch to experiment with CIFAR-10 image classification.
Includes feature map visualization and filter activation analysis.

## Requirements

- Python 3.9+
- PyTorch
- torchvision
- numpy
- matplotlib

It is recommended to use a virtual environment.

Command to install dependencies:
pip install torch torchvision numpy matplotlib

## How to Run
In the project root directory:
python Assignment2.py
